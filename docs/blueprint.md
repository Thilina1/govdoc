# **App Name**: GovDocs LK

## Core Features:

- Centralized Search: Allow users to search for Sri Lankan government and private sector documentation services with a prominent, user-friendly search bar.
- National ID (NIC) Details: Provide detailed information on the 2026 Digital ID process and ICAO photo requirements.
- Passport (K-35 A) Guide: Explain the online passport application process and list regional biometric centers.
- Banking Guide: Provide a guide for digital account opening and personal loans, including Video KYC information.
- e-Locker Information: Inform users about the new 2026 government digital document vault.
- Document Checklist: Provide a checklist of required documents for each service.
- Step-by-Step Tracker: Allow users to track their progress through each service application process.

## Style Guidelines:

- Primary color: Navy blue (#00308F) to convey trustworthiness and professionalism.
- Background color: Very light blue (#F0F8FF), almost white, for a clean and spacious feel.
- Accent color: Crimson (#DC143C) for call-to-action buttons and key interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable interface.
- Simple, clear icons for service categories to enhance user experience.
- Modern, minimalist layout with plenty of white space, adhering to Material 3 design principles.
- Subtle animations for interactive elements to provide feedback without distraction.
